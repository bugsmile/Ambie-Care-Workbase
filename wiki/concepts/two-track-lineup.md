# Two-Track 라인업

> **KSF #3 / GTM Wave 1+2 — 공공과 프리미엄 동시 점유 전략**  
> 출처: [KSF 보고서](../sources/04-ksfs.md), [VPS GTM](../sources/11-vps.md), [가치사슬 분석](../sources/03-value-chain.md), [AOS/DOS](../sources/09-aos-dos.md)

## 전략 개요

단일 제품이 아닌 **공공/제도형 트랙**과 **시설/가족 지불형 트랙**을 분리해 공략한다. 다만 SRS v3.0 기준 MVP는 무료 티어와 1인 개발 제약 때문에 B2B Lite와 B2C 리포트 검증에 먼저 집중한다.

| 구분 | Track A — B2G 공공 보급형 | Track B — B2B 요양시설 프리미엄 |
|-----|--------------------------|-------------------------------|
| **주 고객** | 지자체 (정민석 유형) | 요양시설 (B2B 법인) |
| **채널** | 공공 조달 입찰 | 직접 영업 + EMR 연동 |
| **가격** | 보조금 포함 저가 | ₩100k/월/시설 |
| **하드웨어** | 보급형 모션 센서 | UWB 레이더 풀스펙 |
| **핵심 소구** | 관제 효율 + 예산 절감 | 오탐 제로 + Lock-in |
| **GTM 시기** | 후속 조달/제도 트랙 | Wave 1 B2B Lite 우선 |

---

## 왜 Two-Track인가

### 케어벨 벤치마킹 (가치사슬 분석)
케어벨의 성공 요인: **공공 조달로 레퍼런스 확보 → 프리미엄 B2B 확대** 경로  
→ Rooted도 동일한 레퍼런스 루프 설계

### 시장 세그먼트 근거
- **S1 공공 고위험 재가 안전망:** 약 26.0억원, 전체 SOM의 41.3%
- **S4 장기요양 재가 복지용구:** 약 19.6억원, 전체 SOM의 31.1%
- **S2 가족 지불 홈 안전:** 약 11.5억원, Wave 2 B2C 오픈베타 후보
- **S3 요양시설 안전·운영:** 약 5.9억원, Wave 1 B2B Lite 클로즈드베타 후보

정량 SOM은 S1/S4가 크지만, 초기 실행 가능성은 S3/S2가 높다. 따라서 S3/S2에서 실증 데이터를 만든 뒤 S1/S4로 확장하는 순서가 현실적이다.

## Track A: B2G 공공 보급형

### 타겟
- 지자체 복지 담당자 (정민석 페르소나)
- **예산:** 어르신 1인당 ₩3~5만원/월 서비스 단가 목표

### 핵심 가치
- 방문 요양사 대비 **비용 효율**: 인건비 대체가 아닌 보완
- 야간 응급 사각지대 해소 (독거 어르신 전수 모니터링)
- 조달 KPI: 지자체 1~2곳 레퍼런스 → 전국 복지관 확산

### 제품 스펙 (보급형)
- 기본 모션/적외선 센서 (UWB 제외)
- 관제 대시보드 포함 (신호등 UI, FR-08)
- EMR 연동 불포함. [SRS v3.0](../sources/13-srs-v03.md) 기준 Wave 2 이후 검토

---

## Track B: B2B 요양시설 프리미엄

### 타겟
- 민간 요양시설 원장/운영자
- **예산:** ₩100k/월/시설 (Wave 1), 추후 병상 수 기반 확대

### 핵심 가치
- **Phase 1:** 독립형 신호등 대시보드 → 현장 사용성 검증
- **Wave 2:** EMR 자동 연동 → 이중 기록 제거 → Lock-in
- **Wave 2:** UWB 풀스펙 + Edge AI → 오작동 0.3회/월 목표
- **Wave 2:** 90일+ 데이터 보존 → 법적 분쟁 방어 (장영희 사례)

### Lock-in 구조
케어벨 벤치마킹: 병원 전산망 통합 후 교체 불가  
→ 초기 도입이 곧 **영구 수주** → LTV 극대화

---

## GTM 로드맵

### Wave 1 (클로즈드 베타)
- **대상:** B2B 소형 요양시설 5~10곳
- **목표:** 독립형 대시보드, 데일리 리포트, 오작동 신고 루프 검증
- **수익:** ₩100k × 10시설 = ₩1M/월 (검증 단계)

### Wave 2 (오픈 베타)
- **대상:** B2C 보호자 100~200가구 추가
- **가격:** ₩30k/월 + 설치비 ₩49k
- **목표:** B2C 가격 민감도 검증 + 입소문 채널 확인

### Wave 3 (스케일)
- B2B 전국 확산 + B2G 광역 조달/복지용구 채널 진입
- 두 트랙 데이터 공유 → AI 모델 공통 개선
