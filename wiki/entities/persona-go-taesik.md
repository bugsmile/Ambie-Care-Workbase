# 고태식 (71세) — Non-user 페르소나

> "내가 아직 그런 거 필요 없어"  
> 출처: [페르소나 스펙트럼](../sources/07-persona-spectrum.md), [CJM](../sources/08-customer-journey-map.md)

---

## 프로필

| 항목 | 내용 |
|-----|------|
| **나이/성별** | 71세 / 남성 |
| **직업** | 은퇴 (기초자치단체 6급 공무원, 2019년 명예퇴직) |
| **가족** | 배우자(68세, 중등도 당뇨·고혈압)와 동거 + 자녀 2명 독립 |
| **거주지** | 서울 노원구 36평 자가 아파트 (30년 거주) |
| **건강 상태** | 본인 혈압약 복용, 배우자 인슐린 투약·어지럼증 잦음 |
| **세그먼트** | S2 비활성 — "노인 낙인" 거부형 비사용자 |
| **추정 규모** | **~44~54만 가구** (65~74세 배우자 동거 가구 × "노인 아님" 인식 35%) |

---

## 거부 행동 패턴

장남이 추석에 가져온 것들:
- 스마트워치 → 박스 뜯지 않음 (서랍에 보관)
- 비접촉 케어 서비스 팸플릿 → 쓰레기통
- 스마트플러그 설치 시도 → "빼놔" 지시로 철거

**탐색 차단 사례:**
> 배우자 어지럼증 사건 다음 날, "독거노인 낙상 감지기" 검색  
> 첫 결과 화면에 "시니어 안전 케어 서비스" + 어르신 사진 → 🛑 **검색 창 닫음**

> **핵심 인사이트:** "노인 낙인(Ageism)"이 제품 탐색 행동 자체를 강제 종료시킨다.

## 전환 조건 (가중치 순)

| 전환 조건 | 가중치 | 설계 시사점 |
|---------|-------|----------|
| **"아내를 위한" 프레이밍** | ★★★★★ | "나" → "배우자 보호"로 구매 동기 이동 |
| **제품명에 "노인·시니어·케어" 없음** | ★★★★★ | "홈 안전 센서" / "스마트 공간 모니터" 등 |
| 카메라 없음 명시 | ★★★★☆ | "레이더 전파 = 감시 아님" 설명 자료 필수 |
| 자녀가 아닌 본인이 신청 | ★★★★☆ | 자녀 선물 구조 → 본인 직접 선택 구조 |
| 단지·관리사무소 경유 보급 | ★★★☆☆ | "개인 복지서비스" → "공동주택 안전 인프라" |

## 전환 트리거 시나리오

**Scenario A:** 배우자 실제 낙상 발생 → "아내를 지키는 남편의 선택"으로 포지셔닝  
**Scenario B:** 아파트 입주자 대표 "고독사 예방 차원 스마트홈 안전장치 보급" 발표 → 집합건물 프레이밍으로 개인 낙인 우회

**전환율 78%** — 배우자 건강 이상 시 가장 강력한 전환 트리거

---

## 전환 전후 발화

> **전환 전:** "내가 아직 그런 거 필요 없어."  
> **전환 후 (배우자 어지럼증 재발 후):** "집사람이 욕실에서 또 어지럽다고 하길래... 그냥 하나 달아봤어. 카메라도 아니고 전파같은 거라는데. 나 위한 건 아니고, 집사람 걱정돼서."
