# JTBD 심층 인터뷰 결과

> 원본: `raw/10.jtbd-interview-report.md`

---

## 핵심 검증 가설 (Job-Story)

> **상황:** 독거 부모님 건강 확인을 원하나 웨어러블 거부·충전 방치로 실패한 원격지 자녀  
> **장애물:** CCTV는 프라이버시 거부, 웨어러블은 방치 → 공간적 위급 상황 즉각 인지 불가  
> **기대 결과:** 어르신 조작 0건(Zero-Friction) + 비영상 방식으로 **온전한 심리적 평안(Peace of Mind)**

---

## 3개 그룹별 JTBD 카드

### 그룹 1 — 최근 사용자 (저가 모션센서 경험)
| 항목 | 내용 |
|-----|------|
| **Job Statement** | 부모님 일상생활에 제약 없이 실제 위급 시에만 알림 수신하여 직장에 집중하고자 함 |
| **Desired Outcome** | 오탐 주 1회 이하, 낙상 감지율 100% |
| **핵심 Push** | 낙상 사고 후 극대화된 불안감 |
| **핵심 Pull** | 착용/카메라 없는 비접촉 방식 |
| **핵심 Anxiety** | 오탐 알람으로 인한 알람 피로 → 진짜 위급 시 무감각화 |
| **VoC 증거** | *"이불만 뒤척여도 알람이 와서 이제는 알람이 울려도 무뎌지는 게 더 무섭습니다."* (하루 평균 오탐 12건) |
| **AOS** | **4.0** |

### 그룹 2 — 이탈 경험 그룹 (웨어러블/CCTV 실패)
| 항목 | 내용 |
|-----|------|
| **Job Statement** | 부모님 어떠한 개입도 없이 위급 시 시스템이 능동적으로 보호자와 119에 연결하는 안전망 구축 |
| **Desired Outcome** | 어르신 기기 지속 사용률 100% (Zero-Friction) |
| **핵심 Push** | 기기 착용 없이 외출해 연락 두절 시의 공포 |
| **핵심 Pull** | 한번 설치 후 완전 자동화 환경 |
| **핵심 Habit** | 매일 저녁 "기기 충전하셨냐" 묻다 벌어지는 말다툼 |
| **VoC 증거** | *"비싼 돈 주고 사드렸는데 안 차시면 그만이에요. 제발 아무것도 안 하셔도 되는 기계가 있으면 수백만 원이라도 내겠습니다."* |
| **AOS** | **4.0** |

### 그룹 3 — 미사용 탐색 그룹 (대안 부재)
| 항목 | 내용 |
|-----|------|
| **Job Statement** | 야간 수면·화장실 패턴 변화 데이터로 건강 이상 전조를 조기 방어하고자 함 |
| **Desired Outcome** | 야간 수면/화장실 패턴 오차율 10% 미만, 이상 시 1일 1회 사전 경고 리포트 |
| **핵심 Push** | 부모님 기력 급감 체감 (야간 빈뇨, 수면 저하) |
| **핵심 Pull** | 병원 수준의 예방적 데이터 인사이트 |
| **VoC 증거** | *"밤에 화장실 자주 가시느라 선잠을 주무셔서 우울해하시는 게 더 걱정이에요. 그 패턴만 알 수 있어도 병원에 일찍 모시고 갈 텐데요."* |
| **AOS** | **3.2** |

## Outcome 우선순위표

| Outcome | AOS | 우선순위 |
|---------|-----|---------|
| 오탐 없는 진짜 위급 상황 100% 분별 수신 | **4.0** | Q1 최우선 |
| 어르신 조작 개입 0건 (Zero-Friction 지속성) | **4.0** | Q1 최우선 |
| 카메라 없는 완벽한 실내 동선 파악 | **3.2** | Q1 핵심 |
| 야간 질환 패턴 변화 사전 리포팅 | **3.2** | Q1 핵심 |
| 기기 설치에 대한 어르신 심리적 부담 0화 | 1.8 | Q2 보완 |
