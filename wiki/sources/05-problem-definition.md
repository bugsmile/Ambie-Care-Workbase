# 다관점 문제 정의서

> 원본: `raw/assets/5.Problem-definition.md`  
> 3개 관점(B2B 요양시설, B2G 지자체, B2C 보호자)에서 구조적 미충족(Unmet Need) 상태를 정의

---

## B2B — 요양시설 관점

**대상:** 대규모 실버타운·요양병원 전문 간호 인력 및 관리자

**문제:**
- 기존 저가 모션 센서의 **잦은 오탐**(이불 뒤척임 등) → 업무 피로도 급증
- 낙후된 EMR 전산망과의 **데이터 단절** → 이중 기록 업무
- 오탐 11건 누적 → 알람 무시 → 야간 낙상 사망 (장영희 사례)
- 관내 보급 기기의 **42%가 설치 후 5년 초과**, 허위 출동 연 847건 중 189건이 기기 오탐

**핵심 가치사슬 기회:** 오탐 제로화 AI + EMR 자동 연동 + 직관적 관제 대시보드

---

## B2G — 지자체 관점

**대상:** 한정된 예산으로 돌봄 사각지대 해소해야 하는 복지 공무원

**문제:**
- 고스펙 → 단가 초과, 저스펙 → 응급 징후 누락 → **비용 대 효용의 딜레마**
- 중국산 ODM 화이트라벨링 저가 기기 난립으로 정상 마진 붕괴

**핵심 가치사슬 기회:** 저비용 공공 보급형 + 프리미엄형 투트랙 기획, 공공 조달 파트너십

---

## B2C — 보호자/시니어 관점

**대상:** 독거 부모님을 원격으로 보호하는 자녀

**문제:**
- CCTV: 프라이버시 침해로 상호 거부
- 웨어러블: 충전 방치 → 무용지물
- 결과: **공간적 위급 상황 즉각 인지 방법 없음 → 극도의 불안감 + 데이터 부재**
- CJM 분석: 새벽 오탐 알림 = 최대 위기 지점(P4), 오탐 월 2회 이상 → 해지 의향 급등

**핵심 가치사슬 기회:** Zero-Friction 비접촉 + 비영상 프라이버시 + 데일리 리포트

---

## 3관점 교차 공통 문제

**오탐 문제가 3개 관점 모두에서 핵심:**
- B2B: 오탐 → 알람 피로 → 직원 무시 → 사망 사고
- B2G: 오탐 → 허위 출동 → 예산 낭비 → 조달 기피
- B2C: 오탐 → 신뢰 붕괴 → 구독 해지
