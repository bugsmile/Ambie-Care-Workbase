# 가치사슬 분석 — 케어벨(Carebell) 벤치마킹

> 원본: `raw/assets/3.value-chain.md`

## 핵심 전략 인사이트 3가지

### 1. 오탐(False Alarm) 제로화를 통한 '알고리즘 해자(Moat)' 구축
기기 조립이나 마진 경쟁에 매몰되지 말고, 수집된 데이터를 의료급으로 정제해 내는 **AI 판단 알고리즘 특허 자산화**에 사활.
→ 참조: [오탐률 개념](../concepts/false-alarm.md)

### 2. 웨어러블과의 포지셔닝 차별화 (공간 맥락 지표화)
- '화장실 오가는 횟수(배뇨장애 의심)'
- '특정 구역 체류 지연(낙상 전조)'
→ 참조: [공간 맥락 지표 Moat](../concepts/space-context-moat.md)

### 3. 투트랙(Two-Track) B2B 세일즈 채널
- B2G: 저비용 모션+스마트플러그 조합 엔트리 모델
- B2B 하이엔드 실버타운: UWB 레이더 호흡 센서 프리미엄 납품
→ 참조: [투트랙 라인업](../concepts/two-track-lineup.md)

## 케어벨 가치사슬 단계별 요약

| 단계 | 케어벨 전략 | Rooted 시사점 |
|------|-----------|-------------|
| 핵심 자산 확보 | UWB 레이더 + 수만 명 시니어 생체 파형 데이터 + AI 알고리즘 | 초기 베타 단계에서 데이터 수집 파이프라인 구축 필수 |
| 운영/제품 생산 | 인비저블 설치 (어르신 조작 0) + 오탐 관리 | Zero-Friction 설계 원칙 |
| 고객 전달 | 프리미엄 B2B 주거 일괄 시공 → 접근성 100% | Top-Down B2B2C 모델 |
| 마케팅/영업 | B2G 입찰 회피 + 프리미엄 레퍼런스 세일즈 + Lock-in | 요양원장·건설사 설득 → 전 세대 계약 |
| 고객 서비스 | B2B: EMR SI 기술지원 / B2C: 데일리 리포트 앱 | 앱 데일리 리포트 = 구독 유지 핵심 엔진 |
